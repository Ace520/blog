[GH文件管理器 ](https://ace520.github.io/file-manager) 是已GitHub为存储驱动的web端文件管理器。

历史故事：用vuepress建立了个博客网站，并使用Travis CI做了持续集成，但是每次都需要本地进行文章写作然后提交，不够人性化，就有了GH文件管理器 的诞生。
