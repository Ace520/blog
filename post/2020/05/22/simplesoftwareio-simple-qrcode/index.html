Simple QrCode

Upload files with a single curl command from your terminal! curl --upload-file file.txt https://keep.sh

Introduction

Simple QrCode is an easy to use wrapper for the popular Laravel framework based on the great work provided by Bacon/BaconQrCode (opens new window). We created an interface that is familiar and easy to install for Laravel users.

Official Documentation

Documentation for Simple QrCode can be found on our website. (opens new window)

Contributing

Please submit all issues and pull requests to the simplesoftwareio/simple-qrcode (opens new window) repository on the develop branch!
